# MES全岗位常用术语&行业黑话手册
> 适用：MES实施工程师、需求调研、售前、项目对接，所有缩写附带完整英文名称

## 一、系统架构与上下游系统术语（对接甲方IT高频）
| 完整英文名 | 简称 | 中文释义&核心用法 |
| --- | --- | --- |
| Enterprise Resource Planning | **ERP** | 企业资源计划，顶层计划系统，向下下发生产订单、BOM、领料数据，接收MES完工、消耗回传 |
| Product Lifecycle Management | **PLM** | 产品生命周期管理，存储图纸、版本、完整设计工艺，MES工艺数据来源 |
| Product Data Management | **PDM** | 产品数据管理，PLM子集，中小工厂用于管理BOM与工艺文档 |
| Warehouse Management System | **WMS** | 仓库管理系统，和MES双向同步线边仓、出入库、调拨、盘点库存 |
| Supervisory Control And Data Acquisition | **SCADA** | 设备数据采集监控系统，MES底层数据源，提供设备状态、产量、故障数据 |
| User Acceptance Testing | **UAT** | 用户验收测试，甲方业务人员全流程验证系统，项目验收必备环节 |
| Manufacturing Execution System | **MES** | 制造执行系统，车间生产过程管控核心系统 |
| 无 | 上层系统 | 泛指ERP、PLM/PDM等计划类顶层系统 |
| 无 | 下层设备层 | PLC、机床、传感器、扫码终端等车间硬件采集设备 |
| 无 | 三层架构 | 行业标准分层：ERP计划层 → MES执行层 → SCADA设备采集层 |
| 无 | 接口 | 系统间数据交互通道，分单向推送、双向同步 |
| 无 | 主数据 | 企业共用基础档案：物料、BOM、工序、设备、人员、部门 |
| 无 | 数据中台 | 企业统一数据汇总平台，各系统数据集中存储分析 |

## 二、物料、BOM、工艺类术语（需求调研核心）
| 完整英文名 | 简称 | 中文释义&核心用法 |
| --- | --- | --- |
| Bill of Materials | **BOM** | 物料清单，成品物料构成关系，领料、耗料、成本核算依据 |
| Engineering Bill of Materials | **EBOM** | 设计BOM，PLM输出，纯设计物料结构 |
| Production Bill of Materials | **PBOM** | 生产BOM，MES使用，增加工位、损耗、替代料等生产信息 |
| Serial Number | **SN** | 序列号，一物一码，单品全流程追溯（离散制造） |
| Lot Number | **Lot** | 批次号，整批物料统一编号（化工、食品、原材料） |
| First In First Out | **FIFO** | 先进先出，仓库物料出库规则，防止物料呆滞过期 |
| 无 | 版本 | BOM、物料、工艺迭代编号，生产需锁定正确版本 |
| 无 | 工艺路线 | 产品加工工序顺序、标准工时、作业指导文件 |
| 无 | 工序 | 最小加工步骤：车、铣、装配、检测等 |
| 无 | 工位 | 车间实际作业点位，单台设备/单条产线/班组 |
| 无 | 物料编码 | 物料唯一识别编码，扫码、系统记账基础 |
| 无 | 箱码 | 外包装箱编码，绑定多组SN，批量出入库使用 |
| 无 | 线边仓 | 产线旁临时物料仓，MES独立管控库存 |
| 无 | 投料 | 工单投入原材料，系统校验防错、扣减库存 |
| 无 | 齐套检查 | 开工前校验工单物料是否齐全、库存充足 |
| 无 | 物料防错 | 扫码校验物料编码/批次/版本，杜绝错料生产 |
| 无 | 呆滞料 | 长期积压未流转物料，用于库存周转分析 |

## 三、工单生产执行模块术语（日常实施最高频）
| 完整英文名 | 简称 | 中文释义&核心用法 |
| --- | --- | --- |
| Manufacturing Order | **MO** | 生产工单，MES最小可执行生产任务 |
| Work In Progress | **WIP** | 在制品，车间未完工半成品 |
| 无 | 生产订单 | ERP下发总生产任务，用于拆分工单MO |
| 无 | 工单下达 | 工单下发至工位，允许开工生产 |
| 无 | 报工 | 操作工上报完工数量、工时、不良、设备信息 |
| 无 | 工序流转 | 半成品从上道工序流转至下道工序 |
| 无 | 完工入库 | 工单全部生产完成，成品入库同步WMS/ERP |
| 无 | 工单结案 | 工单核对完毕关闭，不再产生耗料、报工记录 |
| 无 | 超耗 | 实际物料消耗＞BOM标准用量 |
| 无 | 结余 | 工单剩余原材料，需退料入库 |
| 无 | 拆分工单 | 大单拆多张小工单分配不同产线 |
| 无 | 合并工单 | 同规格工单合并生产，减少换模损耗 |

## 四、质量管理模块术语
| 完整英文名 | 简称 | 中文释义&核心用法 |
| --- | --- | --- |
| Incoming Quality Control | **IQC** | 来料检验，原材料入库质检 |
| In-Process Quality Control | **IPQC** | 过程巡检，生产中途抽样检测 |
| Final Quality Control | **FQC** | 完工终检，工序完成后成品检验 |
| Outgoing Quality Control | **OQC** | 出库检验，发货前最终核查 |
| 无 | 不良品 | 不符合质检标准的产品 |
| 无 | 不良原因代码 | 标准化缺陷分类编码，用于质量统计分析 |
| 无 | 返工 | 不良品回流工序重新加工，二次质检 |
| 无 | 报废 | 产品无法修复，做报废扣料处理 |
| 无 | 让步接收 | 轻微缺陷特批放行入库/发货 |
| 无 | 质量追溯 | 通过SN/LOT反向查询物料、人员、设备、时间、检验记录 |

## 五、设备采集模块术语（设备对接必备）
| 完整英文名 | 简称 | 中文释义&核心用法 |
| --- | --- | --- |
| Programmable Logic Controller | **PLC** | 可编程逻辑控制器，自动化设备控制核心 |
| Open Platform Communications Unified Architecture | **OPC-UA** | 新一代工业通讯协议，主流设备采集标准 |
| Open Platform Communications Data Access | **OPC-DA** | 老式Windows工业通讯协议，逐步淘汰 |
| Distributed Numerical Control | **DNC** | 机床联网系统，程序下发、机床数据采集 |
| Overall Equipment Effectiveness | **OEE** | 设备综合效率，车间核心KPI：可用率×性能率×良品率 |
| 无 | 点位 | PLC内部数据地址，存储产量、状态、故障数值 |
| 无 | 自动采集 | MES直连设备自动抓取数据，无需人工录入 |
| 无 | 手动采集 | PDA扫码、电脑手工填报产量工时 |
| 无 | 设备台账 | 设备基础档案：型号、维保周期、负责人 |
| 无 | 停机原因 | 故障、待料、换模、休息、计划停机分类 |
| 无 | 心跳采集 | MES定时轮询设备，检测在线状态，断线告警 |

## 六、实施项目管理&配置黑话（写方案、内部沟通）
| 完整英文名 | 简称 | 中文释义&核心用法 |
| --- | --- | --- |
| 无 | 主数据初始化 | 项目上线前批量导入物料、BOM、人员、设备档案 |
| 无 | 模板导入 | Excel标准模板批量导入数据，替代手工新增 |
| 无 | 流程配置 | 配置工单审批、工序流转、单据审批流 |
| 无 | 权限矩阵 | 各岗位角色操作权限清单 |
| 无 | 单据模板 | 自定义领料单、报工单、质检单打印格式 |
| 无 | 试点上线 | 单条产线先行验证流程，稳定后全厂推广 |
| 无 | 分批上线 | 分车间、分模块逐步启用MES |
| 无 | 补录数据 | 上线前历史生产、库存数据补录进系统 |
| 无 | 数据清洗 | 整理重复、错误、残缺物料/BOM基础数据 |
| 无 | 二次开发 | 标准功能无法满足客户，定制代码开发 |
| 无 | 看板可视化 | 车间大屏，展示产量、OEE、不良率、达成率 |
| 无 | 上线切换 | 停用手工台账/旧系统，正式全量使用MES |
| 无 | 运维 | 上线后问题排查、备份、升级、用户培训 |
| 无 | 验收 | 甲方确认项目交付完成，签署验收单回款 |

## 七、报表与生产指标术语
| 完整英文名 | 简称 | 中文释义&核心用法 |
| --- | --- | --- |
| 无 | 达成率 | 实际产量 ÷ 计划产量，考核生产任务完成情况 |
| 无 | 生产节拍 | 单位时间产出量，衡量产线速度 |
| 无 | 物料消耗报表 | 工单标准耗料、实际耗料、超耗结余明细 |
| 无 | 工序追溯报表 | 单品全工序流转、人员、设备、质检完整记录 |
| 无 | 稼动率 | 设备实际运行时长 ÷ 计划开机时长，设备利用率指标 |
